# Estilo Reportes Sintéticos

**Fecha:** 5 Mar 2026
**Feedback:** Wally prefiere reportes más sintéticos

## ✅ NUEVO ESTILO PARA FINANZAS

### ❌ Evitar (demasiado verbose):
- Explicaciones largas del sistema
- Repetir datos ya conocidos
- Secciones innecesarias 
- Múltiples ✅ ❌ 🔴 excesivos
- "RESUMEN FINANCIERO COMPLETO CON..." etc.

### ✅ Preferir (sintético):
- Datos clave directo al punto
- Solo lo nuevo/importante
- Formato bullet simple
- Confirmar acción + próximo paso

## 📋 TEMPLATE SINTÉTICO FINANZAS

```
📄 **[BANCO] [TARJETA] PROCESADO**

**Datos clave:**
- Monto: $X.XXX.XXX
- Vence: DD/MM
- Registrado ✅

**Pendientes actualizados:** $X.XXX.XXX

**Próximo:** [acción necesaria]
```

## 🎯 APLICAR A:

- Resúmenes de tarjetas
- Procesamiento de pagos  
- Updates de sistema
- Reportes de estado

**REGLA:** Menos explicación, más acción directa.
